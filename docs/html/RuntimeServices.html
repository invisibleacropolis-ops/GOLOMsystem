Class: RuntimeServices (inherits from Node)

Brief Description

Aggregates core logic modules and exposes them to scenes. This node wires together the grid map, turn manager, and supplemental services (attributes, statuses, abilities, loadouts, reactions, and event bus) so gameplay scenes can access a single entry point for logic state.
